# Uptime DPin - WebSocket Setup

## Quick Start

### 1. Install dependencies

```bash
npm install
cd apps/hub && npm install
cd ../validator && npm install
```

### 2. Start Hub

```bash
cd apps/hub
npm run dev
```

### 3. Start Validator

```bash
cd apps/validator
npm run dev
```

See [WEBSOCKET_ARCHITECTURE.md](../../WEBSOCKET_ARCHITECTURE.md) for full documentation.
